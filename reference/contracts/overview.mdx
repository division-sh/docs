Contracts are validated by the static analyzer before the runtime boots, so this section is
effectively Swarm's language reference. Each page documents one contract file: every field,
its type, and its constraints. This reference tracks platform specification **v1.6.0**.

For the authoritative source, see `platform-spec.yaml` in the Swarm repository. Where this
reference and the spec disagree, the spec wins.
